# Chef's Menu App

## Setup Instructions

1. Extract the zip file
2. Open terminal in the project folder
3. Run: `npm install`
4. Run: `npx expo start`
5. Press `w` to open in web browser

## How to Use

- **Home Screen**: View menu stats and navigate
- **Add Dish**: Create new menu items with images
- **Menu List**: View and manage all dishes
- **Filter**: Filter dishes by course type
- **Full Menu**: Customer-facing menu display

## Image Upload

- On mobile: Tap image area to choose from gallery
- On web: Use file picker or paste image URL
